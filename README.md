## Commands

### 安装 pnpm

npm i pnpm -g

### 安装所有依赖

pnpm install

### 本地测试

pnpm run exm:dev

### 打包

pnpm run build:kitty

### 发布

pnpm run publish:kitty

### 查看测试覆盖率
